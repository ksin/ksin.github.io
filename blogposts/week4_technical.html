Enumerable#map evalutes every element in a collection through a block and returns a new collection with the evaluated elements inside.
ie: x = [1,2,3,4]
x.map {|num| num + 3} #=> [4,5,6,7]

Enumerable#group_by evalutes every element in a collection through a block and returns a hash where the key is the evaluated result and the values are arrays of elements in the collection that correspond to that result.
ie: (1..6).group_by { |i| i%3 } #=> {0=>[3, 6], 1=>[1, 4], 2=>[2, 5]}

Enumerable#cycle calls the block for each element in the collection forever, or if there is an argument, repeat for the argument's number of times.
ie: a = ["a", "b", "c"]
a.cycle { |x| puts x } # print, a, b, c, a, b, c,.. forever.
a.cycle(2) { |x| puts x } # print, a, b, c, a, b, c.
